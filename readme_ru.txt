	Назначение проекта.

Это утилита для отправки писем электронной почты по протоколу SMTP. Её удобно 
использовать в работе инженеров DevOps для проверки работы электронной почты в различных средах. 
Можно вводить и менять следующие параметры:
	○ Username
	○ Password
	○ Host
	○ Port
	○ Use SSL
	○ Use default credentials
	○ From
	○ To
	○ Subject
	○ Body
Позволяет подменить адрес отправки, для этого следует указать желаемый адрес в поле From.


	Особенности реализации.
	
○ Приложение реализовано с использованием шаблона проектирования MVVM.
○ Приложение использует .NET Framework 4.7.2.
